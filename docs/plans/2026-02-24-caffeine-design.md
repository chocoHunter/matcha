# Caffeine - macOS 菜单栏防休眠工具

**项目**：Caffeine
**类型**：macOS 菜单栏工具
**版本**：1.0.0
**日期**：2026-02-24

---

## 1. 项目概述

Caffeine 是一款 macOS 菜单栏应用，通过阻断系统休眠让 Mac 保持清醒。用户可以灵活控制多种防休眠模式，咖啡因主题设计直观有趣。

## 2. 目标用户

- 需要长时间运行任务的用户（渲染、下载、编译）
- 外接显示器工作需要合盖运行的用户
- 希望自定义电源管理的高级用户

## 3. 功能列表

### 3.1 防休眠模式

| 模式 | 说明 | 命令 |
|------|------|------|
| 清醒模式 | 仅禁止系统睡眠 | caffeinate -i |
| 屏幕常亮 | 禁止屏幕关闭 | caffeinate -d |
| 极致模式 | 合盖也能运行 | caffeinate -i -s |
| 定时模式 | 设置 X 分钟后自动恢复 | caffeinate -i -t 3600 |

### 3.2 电源智能恢复

- 电池低于 X% 自动恢复休眠（默认 20%）
- 接通电源时允许一直保持
- 可自定义恢复百分比（10%/20%/30%/50%）

### 3.3 系统集成

- 开机自动启动（可选开关）
- 菜单栏电池百分比显示
- 当前状态实时显示

### 3.4 退出功能

- 一键退出应用
- 退出时自动恢复默认休眠

## 4. UI/UX 设计

### 4.1 菜单栏图标

| 状态 | 图标 | 说明 |
|------|------|------|
| 正常 | ☕ 咖啡杯（空） | 未开启防休眠 |
| 清醒模式 | ☕ 咖啡杯（半满） | 仅禁止睡眠 |
| 屏幕常亮 | ☕ 咖啡杯（满） | 屏幕常亮 |
| 极致模式 | ⚡ 咖啡杯（发光） | 合盖运行中 |
| 定时模式 | ⏱️ 咖啡杯（带计时器） | 倒计时中 |

### 4.2 菜单结构

```
[图标] ─────────────────────────────────────
│
│  当前状态：☕ 清醒模式 (已开启 00:15:32)
│  电池：85% 🔌
│  ───────────────────────────────────────
│
│  ○ 清醒模式          [开启/关闭]
│  ◉ 屏幕常亮
│  ○ 极致模式
│  ───────────────────────────────────────
│
│  ⏱️ 定时: [15分钟 ▼]
│  ───────────────────────────────────────
│
│  🔋 电池低于 [20% ▼] 自动恢复
│  ☐ 开机自动启动
│  ───────────────────────────────────────
│
│  ❌ 退出 Caffeine
└───────────────────────────────────────────
```

### 4.3 交互逻辑

- **点击图标** → 展开菜单
- **选择模式** → 立即生效，图标变化
- **开启电源保护** → 低于设定值自动关闭
- **退出** → 恢复系统默认休眠状态

## 5. 技术架构

### 5.1 技术栈

- **语言**：Swift 5.9+
- **框架**：AppKit（菜单栏）+ Foundation
- **依赖**：无外部依赖，纯原生实现
- **最低系统**：macOS 12.0 (Monterey)

### 5.2 核心模块

| 模块 | 职责 |
|------|------|
| `AppDelegate` | 应用入口，菜单栏初始化 |
| `StatusBarController` | 菜单栏图标和菜单管理 |
| `CaffeinateManager` | caffeinate 进程控制 |
| `PowerManager` | 电池状态监听，自动恢复 |
| `PreferencesManager` | 用户设置存储（UserDefaults） |

### 5.3 数据流

```
用户点击 → StatusBarController → CaffeinateManager → 执行 caffeinate
                                      ↓
                              PowerManager 监听电池
                                      ↓
                              低电量 → 自动停止 → 恢复休眠
```

## 6. 错误处理

| 场景 | 处理 |
|------|------|
| caffeinate 执行失败 | 弹出提示，菜单显示错误状态 |
| 电池信息读取失败 | 隐藏电池相关功能 |
| 用户手动关闭 | 自动停止 caffeinate 进程 |
